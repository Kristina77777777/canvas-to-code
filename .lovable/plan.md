

# Magic Coffee Shop & Art Space Website
*# Сайт "Мэджик Кофейня & Арт-пространство"*

## Project Overview / Обзор проекта

A modern coffee shop website with an art space in a coastal/ocean style. Soft sea colors, elegant typography, and smooth animations create a relaxing beach atmosphere.

*Современный сайт кофейни с арт-пространством в морской стилистике. Мягкие океанские цвета, элегантная типографика и плавные анимации создают атмосферу прибрежного релакса.*
